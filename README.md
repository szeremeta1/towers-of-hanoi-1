# Towers of Hanoi

A Java program that lets you play and visualize the classic Towers of Hanoi puzzle. The goal is to move all the disks from one peg to another, one at a time, without ever placing a larger disk on top of a smaller one.

## How to Run

1. Compile: `javac *.java`
2. Run the GUI: `java HanoiGUI`

## Features

- Interactive GUI where you can click pegs to move disks
- Choose the number of disks
- Auto-solve button that animates the solution step by step
- Uses recursion to calculate the optimal solution
